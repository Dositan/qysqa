About Qysqa

What on the Earth is qysqa?

Qysqa is an open-source URL shortener project.

It was created using Flask + Jinja templating system.

Put URL → Shorten → Use

Features:

  • Custom URL token
  • Short URL Clicks Tracker
  • URL Lookup

Plans:

  • Implement Authentication System
  • Public/Private Short URLs
  • Qysqa User Profiles
